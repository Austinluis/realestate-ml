# Templates

This directory contains Django templates for the server-rendered web interface.

## Structure

- `base.html`: shared layout, navigation, messages, footer, CSS/JS includes.
- `accounts/`: login and registration pages.
- `dashboard/`: authenticated dashboard page.
- `properties/`: property list, detail, add, and edit pages.
- `predictions/`: prediction form and prediction history pages.

Templates use Bootstrap classes, Bootstrap Icons, and the custom design system in `static/css/style.css`.
